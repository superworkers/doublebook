# Haven — AI Assistant for Real Estate Agents

Landing page for early waitlist signups.

## View Locally

```bash
# Option 1: Python (comes with macOS)
python3 -m http.server 8000

# Option 2: Node.js
npx serve
```

Then open http://localhost:8000

## Deploy to GitHub Pages

1. Create a new repo on GitHub
2. Push this folder
3. Go to Settings → Pages → Deploy from main branch
4. Your site will be live at `https://yourusername.github.io/repo-name`

## Structure

```
├── index.html    # Single-file landing page (HTML + CSS + JS)
└── README.md     # This file
```

## Customization

- **Brand name**: Search for "Haven" and replace
- **Colors**: Edit CSS variables in `:root` at the top of the `<style>` block
- **Form**: Connect to your email service (Mailchimp, ConvertKit, etc.)
